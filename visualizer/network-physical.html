物理层

比特流传输与带宽、奈奎斯特/香农上限

考点速记

  • 四特性：机械、电气、功能、过程
  • 视角：时域与频域，频域更直观
  • 傅里叶：复杂周期信号可分解为多正弦叠加
  • 带宽：理论无限；有效带宽受物理信道衰减限制
  • 奈奎斯特（无噪声）：C = 2·H·log₂(V)
  • 香农（有噪声）：C = H·log₂(1 + S/N)；S/N = 10^{dB/10}
  • 综合：最大速率为两者的较小值
  • 传输方式：模拟/数字，串行/并行，单工/半双工/全双工

速算工具

奈奎斯特
C = 2·H·log₂(V)
香农
C = H·log₂(1 + S/N)
分贝换算
S/N = 10^{dB/10}
最终速率
C = min(C_Nyquist, C_Shannon)
