## 数据结构

一、判断题

1. $\log^{*} n = O(\log^{*}(\log n))$
tags: [迭代对数log*]
知识点: 迭代对数
答案: 正确
解析:
  记以 2 为底的迭代对数：\(\log^{*} n = 1 + \log^{*}(\log_2 n)\)（当 \(n>1\)）。因此对于任意 \(n>1\)，有
  \[\log^{*} n = 1 + \log^{*}(\log n)\]
  故 \(\log^{*} n\le 2\,\log^{*}(\log n)\)（当 \(\log^{*}(\log n)\ge 1\)）。存在常数 \(c=2\) 与阈值 \(n_0\)，使得对所有 \(n\ge n_0\)，\(\log^{*} n \le c\,\log^{*}(\log n)\)。因此 \(\log^{*} n = O(\log^{*}(\log n))\)。
2. 起泡排序中存在某元素的起始位置与其最终位置相邻,甚至已经处于最终位置,却需要参与 $n-1$ 次交换的情况 
3. 对规模为 $n$ 的 AVL 树做一次插入操作,最坏情况下可能引发 $\Omega(\log n)$ 次局部重构 
4. 在一棵二叉树中,任一节点的关键码值都大于它的(如果存在)左孩子的关键码,且小于它的右孩子(如果存在)的关键码值,则此二叉树一定是一棵二叉排序树 
5. 只有在访问序列具有较强的局部性时,伸展树才能保证分摊 $O(\log n)$ 的性能。 
6. 相对于同样规模的完全二叉堆,多叉堆上滤成本会降低,下滤成本会上升。 
7. 字符集越大 KMP 算法和蛮力算法比越没有优势 
8. 在插入操作后若红黑树黑高度增加,则在双红修复过程中仅做过重染色,而无任何结构调整。
