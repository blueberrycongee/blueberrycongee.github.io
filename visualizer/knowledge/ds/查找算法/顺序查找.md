# 顺序查找

## 1. 基本概念

顺序查找（Sequential Search）是最简单的查找算法，从数据结构的一端开始，逐个比较，直到找到目标元素或遍历完所有元素。

## 2. 核心特点

- **无序列表适用**：不要求数据有序
    
- **实现简单**：最基础的查找方法
    
- **效率较低**：需要逐个比较
    

## 3. 时间复杂度

- **最好情况**：O(1) - 第一个元素就是目标
    
- **最坏情况**：O(n) - 最后一个元素或不存在
    
- **平均情况**：O(n) - 平均需要比较 n/2 次
    

## 4. 代码

普通版本

```c++
int sequentialSearch(const vector<int>& arr, int target) {
    for (int i = 0; i < arr.size(); ++i) {
        if (arr[i] == target) {
            return i;
        }
    }
    return -1;
}
```

哨兵法优化

```c++
int sentinelLinearSearch(vector<int>& arr, int target) {
    int lastElement = arr.back();
    arr.back() = target;
    
    int i = 0;
    while (arr[i] != target) {
        i++;
    }
    
    arr.back() = lastElement;
    
    return (i != arr.size() - 1 || arr.back() == target) ? i : -1;
}
```

## 5. 优化技巧

1. **哨兵优化**：在数组末尾添加目标值作为哨兵，省去边界检查
    
2. **概率优化**：将经常查找的元素放在前面（自组织查找）
    

---

## 6. 适用场景

- 数据量较小
    
- 数据无序且不便排序
    
- 数据结构为链表（无法随机访问）
