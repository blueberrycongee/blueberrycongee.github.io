# 迭代对数（log*）

迭代对数通常以 2 为底定义为：

> \[\log^{*} n = \begin{cases} 0 & (n \le 1) \\ 1 + \log^{*}(\log_2 n) & (n > 1) \end{cases}\]

性质：

- \(\log^{*} n\) 的增长极其缓慢，通常小于 6。
- 对于任意 \(n>1\)，有 \(\log^{*} n = 1 + \log^{*}(\log n)\)。

在算法分析中，常见到如下渐近关系：

> \(\log^{*} n = O(\log^{*}(\log n))\)

这可以通过上面的递归定义与常数放缩得到。

> [!summary] 提示
> 许多使用分层迭代减少规模的算法（例如某些并查集的路径压缩分析）会出现 \(\log^{*}\) 这样的极慢增长项。
